# 🐈‍⬛ Git & GitHub Notes

This repository contains my **Git and GitHub notes**.

All topics are explained using **commands, examples, and screenshots**, not just theory.

## What this repo contains

This repository is divided into **5 modules**:

```
git-github-notes
│
├── 01-introduction-setup
├── 02-git-basics-local
├── 03-workflow-clone-commit-push
├── 04-git-init-lifecycle
└── 05-branches-and-practices
```

Each folder contains:
- A README.md file
- Git commands with explanation
- Real command outputs
- Screenshots for better clarity

## What you will learn

By going through this repository, you will learn:
- What Git and GitHub are
- How Git works on your local machine
- How to create and manage repositories
- How to add, commit, and push code
- How GitHub is used in real projects
- How branching works in Git

## How to use this repo

1. Start from the first folder
2. Read the README.md
3. Try the commands on your system
4. Check the outputs
5. Move to the next module

This repo is meant to be **learn by doing**, not just reading.

## Tools used

- Git
- GitHub
- Linux terminal
- Git Bash
